Aplicația frontend (client):
1. Se instalează NodeJS: https://nodejs.org/en/download/
2. În consolă se introduce comanda:
	npm install -g @angular/cli
3. Se navighează în folderul frontend/licenta-front
4. În consolă se introduce comanda:
	npm install
5. Aplicația se rulează folosind comanda:
	ng serve
